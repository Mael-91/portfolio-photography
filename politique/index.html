Politique de confidentialité

Dernière mise à jour : 11 février 2026
Cette page explique comment les données personnelles sont traitées lorsque vous contactez Mael Constantin, photographe automobile freelance, via le formulaire de contact/devis.

1. Responsable du traitement

Le responsable du traitement est : Mael Constantin (photographe automobile freelance).
Contact : mael.constantin@gmail.com

2. Données collectées

Selon le type de demande (professionnel, particulier, information), le formulaire peut collecter :

  • Prénom et nom
  • Adresse email
  • Numéro de téléphone
  • Société (pour les demandes professionnelles)
  • Durée estimée de mission (si renseignée)
  • Message libre
  • Préférence de contact par téléphone (case à cocher)

3. Finalité et base légale

Les données sont utilisées uniquement pour :

  • répondre à votre demande de contact ou de devis ;
  • échanger avec vous sur le projet ;
  • établir une proposition commerciale si vous en faites la demande.

Base légale : mesures précontractuelles (réponse à une demande de devis) et/ou intérêt légitime (réponse à une demande d’information). Les références CNIL rappellent que sur un site vitrine, il faut informer clairement les personnes, notamment au niveau du formulaire.

4. Caractère obligatoire ou facultatif

Les champs marqués comme obligatoires dans le formulaire sont nécessaires pour traiter votre demande. Si vous ne les renseignez pas, la demande peut ne pas être traitée.

5. Destinataires

Les données sont destinées uniquement à Mael Constantin pour répondre à votre demande. Elles peuvent transiter et être stockées via l’hébergeur/plateforme de formulaires : Netlify (sous-traitant).

6. Durée de conservation

Les données sont conservées :

  • jusqu’à 12 mois après le dernier échange pour une demande sans suite ;
  • le temps de la relation commerciale si une prestation est réalisée, puis archivées selon les obligations légales applicables.

7. Sécurité

Des mesures raisonnables sont mises en place pour protéger les données (HTTPS, accès limité). Le site est hébergé sur Netlify.

8. Vos droits (RGPD)

Vous disposez notamment des droits d’accès, de rectification, d’effacement, d’opposition et de limitation. Pour exercer vos droits, contactez : mael.constantin@gmail.com.

9. Cookies

Ce site n’utilise pas de cookies publicitaires. S’il utilise des cookies de mesure d’audience à l’avenir, cette page sera mise à jour et un mécanisme adapté sera mis en place.
